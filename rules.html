Follow the rules

List of all Minecraft and Discord server rules.

Ignorance of the rules is no excuse!

Minecraft server

  1. tidak diperbolehkan menggunakan toolbox/cheat
  2. tidak diperbolehkan menggunakan xray
  3. tidak diperbolehkan Toxic
  4. bagi kaum suki diserver akan diperlakukan sangat buruk
  5. jangan meminta barang atau apapun dengan admin
  6. jangan membuat keributan didalam server
  7. pertarungan antarclan jika ada salah satu yg melakukan hal yg dilanggar maka akan dibanned permanen
  8. member jangan pernah menghancurkan block didekat lobby!!

Discord server

  1. Jangan merusak fasilitas discord
  2. Pembulian akan terdeteksi oleh bot
  3. jangan mengirimkan foto/video/kata kata yg ke18+
  4. jangan sering sering mengetag admin
  5. jangan menghancurkan jati diri server
  6. Admin tdk diperbolehkan membuli member
  7. tdk diperbolehkan toxic
  8. tidak diperbolehkan perkelahian antarperkataan

More rules

  1. admin jangan pernah memberi barang kemember
  2. admin tidak diperbolehkan menggunakan kuasanya sementara menta
  3. admin tidak diperbolehkan toxic/apapun itu

  5. admin tidak diperbolehkan merusak server
  6. admin harus membantu member baru
  7. admin harus selalu mengoptimalkan kinerja server

If you have read the rules, you can go and enjoy the fun on our server!
